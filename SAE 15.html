Projet

Nous allons vous présenter les résultats de ce projet sur ce site. L'objectif de cette sae était de traiter des données, pour cela avec mon binôme, nous avons d'abord utiliser le site open data montpellier afin de récupérer les données des parkings de voitures pour ensuite les analyser et les traiter sous forme de programme python. Nous avons fais la même chose pour les parkings vélos, et nous avons créer un prgramme qui va permettre de calculer le taux d'occupation des ces parkings. Par la suite afin de réaliser une corélation entre les deux parkings nous avons utiliser le logiciel GnuPlot pour obtenir un graphique et savoir si il le taux d'occupation des parkings vélos ont une relation avec l'usage de la voiture.
