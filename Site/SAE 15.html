Projet:

Je vais vous présenter les résultats de ce projet sur ce site. L'objectif de cette sae était de traiter des données, pour ce fire, j'ai utiliseé le site open data montpellier afin de récupérer les données des parkings de voitures pour ensuite les analyser et les traiter sous forme de programme python. J'aicrée différents programmes que vous pouvez retrouver dans mon compte rendu qui permettent de calculer le taux d'occupation des ces parkings. Par la suite afin de réaliser le traitement de ces données, j'ai utilisé le logiciel GnuPlot pour obtenir un graphique et analysé selon différents jours.
